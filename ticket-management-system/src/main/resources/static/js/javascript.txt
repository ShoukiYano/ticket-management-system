DOMContentLoadedイベントリスナー:
ドキュメントの読み込みが完了したときに実行される関数を設定します。
フォーム、ハンバーガーメニュー、サイドバー、サイドバーの閉じるボタンを取得します。
フォーム送信時にバリデーションを行うイベントリスナーを設定します。
ハンバーガーメニューのクリックでサイドバーを表示し、閉じるボタンのクリックでサイドバーを非表示にするイベントリスナーを設定します。

validateForm関数:
フォームのバリデーションを行う関数です。
必須フィールドが空でないかをチェックし、空の場合はエラーメッセージを表示します。
全ての必須フィールドが入力されていれば true を返し、そうでない場合は false を返します。

showError関数:
エラーメッセージを表示する関数です。
既存のエラーメッセージがある場合はそれを更新し、ない場合は新しいエラーメッセージを作成して表示します。

clearError関数:
エラーメッセージをクリアする関数です。
既存のエラーメッセージがある場合はそれを削除し、入力フィールドからエラークラスを削除します。
